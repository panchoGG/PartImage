# PartImage

part镜像是一个专为逆向分析而生的安卓13系统、具备脱壳、定制抓包、定制frida等功能，

目前支持pixel 4/4XL 以及 pixel 6。

定制的联系QQ：328366802

## 环境

环境干干净净，自带内核级定制root，支持谷歌框架，在应用看来就是一台普通的手机。你不再需要为root检测、xposed检测、vpn检测、ksu检测、面具等等检测而烦恼。
